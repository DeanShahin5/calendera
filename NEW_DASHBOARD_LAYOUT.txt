This file contains the new collapsible accordion layout structure for the dashboard.

The new design replaces the filter-based system with an always-visible collapsible accordion where:
- Each category has a header with count and chevron arrow
- Colored vertical bar on the left
- Click to expand/collapse
- Content shows when expanded

Structure:
<div className="space-y-4">
  {/* Events Section */}
  <div className="bg-surface/50 border border-border/50 rounded-xl overflow-hidden">
    <button
      onClick={() => toggleSection('events')}
      className="w-full p-4 flex items-center gap-4 hover:bg-foreground/5 transition-colors"
    >
      <div className="w-1 h-10 rounded-full bg-gradient-to-b from-[rgb(66,133,244)] to-[rgb(52,168,83)]"></div>
      <div className="flex-1 text-left">
        <h2 className="text-2xl font-bold text-foreground">{events.length} Events</h2>
      </div>
      <svg
        className={`w-6 h-6 text-foreground/60 transition-transform ${expandedSections.has('events') ? 'rotate-180' : ''}`}
        fill="none"
        viewBox="0 0 24 24"
        stroke="currentColor"
      >
        <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M19 9l-7 7-7-7" />
      </svg>
    </button>

    {expandedSections.has('events') && (
      <div className="p-4 border-t border-border/50">
        {/* Existing events content */}
      </div>
    )}
  </div>

  {/* Repeat for Tasks, Social, Spam, Recruitment, Messages */}
</div>

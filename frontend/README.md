# Frontend

Next.js 16 + React 19 + TypeScript web application.

## Setup

```bash
npm install
```

## Environment Variables

Create `.env.local`:

```bash
ANTHROPIC_API_KEY=your_api_key_here
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm run start
```
